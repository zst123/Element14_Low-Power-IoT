Shield Library (CY8CKIT-028-TFT)
API Reference
The following provides a list of API documentation
 ShieldBasic set of APIs for interacting with the CY8CKIT-028-TFT shield board
 PinsPin mapping of the GPIOs used by shield peripherals
 MicrophoneThe microphone uses the HAL PDM_PCM driver for its initialization
 Audio CodecThe audio codec is handled by the audio-codec-ak4954a library, details are available at https://github.com/cypresssemiconductorco/audio-codec-ak4954a
 Light SensorThe light sensor is handled by the sensor-light library, details are available at https://github.com/cypresssemiconductorco/sensor-light
 Motion SensorThe motion sensor is handled by the sensor-motion-bmi160 library, details are available at https://github.com/cypresssemiconductorco/sensor-motion-bmi160
 TFT DisplayThe display is handled by the display-tft-st7789v library, details are available at https://github.com/cypresssemiconductorco/display-tft-st7789v
