Hardware Abstraction Layer (HAL)
HAL Driver Availability

General Description

This section documents the macros that can be used to check if a specific driver is available for the current device.

This includes macros for all HAL drivers that exist for any device, even those that are not applicable (included) with this library. This is intended to help write code that port between device categories that might not have the same drivers available.
